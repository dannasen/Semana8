3. Contenido para `buenas-practicas.txt`

4. Contenido para `seguridad-plataformas.txt`
1. No exponer datos personales de los usuarios en áreas públicas de la plataforma.
2. Proteger credenciales usando encriptación y autenticación de dos factores.
3. Usar conexiones seguras HTTPS para que la información viaje cifrada.
4. Registrar accesos y acciones de los usuarios para detectar actividades sospechosas.
5. Limitar acceso a información sensible solo al personal autorizado que lo necesite.
---

5. Texto para `reflexion-semana8.txt`

8-10 líneas sobre: _¿Por qué la seguridad digital es importante para proteger plataformas tecnológicas que usan comunidades o procesos participativos?_
La seguridad digital es fundamental en plataformas comunitarias porque protegen 
los datos y la confianza de las personas que participan. Si una plataforma sufre 
un ataque, se pueden filtrar datos personales, contraseñas y conversaciones privadas, 
lo que daña a los usuarios y a la comunidad.

Además, sin seguridad los procesos participativos pierden validez. Si alguien 
puede alterar votos, comentarios o propuestas, la plataforma deja de ser un 
espacio democrático y confiable.

También es importante porque las comunidades dependen de la participación activa. 
Si los usuarios sienten que su información no está segura, dejan de usar la plataforma 
y el proyecto fracasa. Por eso, aplicar buenas prácticas como contraseñas fuertes, 
accesos limitados y actualizaciones constantes es clave para mantener un ambiente 
seguro, inclusivo y participativo donde todos puedan aportar sin miedo.
